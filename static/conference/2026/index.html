Amplifyで事業を前に進めるためのヒントがここに

Amplify Japan User Groupが主催する年に一度のカンファレンス。Amplifyをまだ採用していないチームや、新規事業の立ち上げを加速させたいリーダーに向けて、 実践的な知見とコミュニティとのつながりを提供します。午前は初心者向けハンズオンで「Amplifyの第一歩」を体験し、午後は企業での活用事例やPoCから事業化までの道のりを共有します。

PoCから本番までの最短距離 Amplifyで実現する高速立ち上げ 初期コスト最適化とガバナンス マイグレーションコストの低減 OSSコミュニティとの連携 持続可能なプロダクト開発

対象となる方

  • 新規事業開発・DX推進のリーダー
  • スピーディなPoCを求めるプロダクトオーナー
  • AWS活用を部署に広げたいテックリード
  • Amplify未経験だが検討中のエンジニア・PM

得られる価値

  • Amplify導入の意思決定に役立つ成功・失敗事例
  • PoCと本番を橋渡しするアーキテクチャと運用ノウハウ
  • 社内提案を後押しする事例や情報
  • コミュニティを通じた継続的なナレッジシェア

プログラム・ハイライト

ハンズオン

AmplifyもAWSも初めての方向けに、サービス構築のワークショップを実施。ゼロからアプリを公開するまでの流れを体験します。

30名限定

キーノート

企業における新規事業部門でのAmplify活用事例を紹介。PoCから事業化までを加速させた意思決定とガバナンスの工夫を深掘りします。

注目

登壇セッション

複数の企業・開発チームからのケーススタディ。失敗談や移行コスト、OSSコントリビュートの実例などを交え、Amplify導入の実際を語ります。

コミュニティメンバーやAWS SAと直接つながるネットワーキング。具体的な導入相談やマッチングの場としてご活用ください。
